# 08. 同类 App 功能覆盖矩阵

目标：把 Nook X、NotchNook、NotchBox 以及用户提到的 PopX / ShoX 这类 Mac 顶部效率岛产品，拆成可开发的功能池。

PopX / ShoX 暂无足够稳定公开资料，因此先按同类产品能力覆盖。后续如果拿到截图或链接，再逐项补齐。

## 覆盖层级

- P0：先做出来并能日用。
- P1：接近同类 App 主功能覆盖。
- P2：增强沉浸体验和高级系统集成。
- P3：可玩性、自定义和插件化。
