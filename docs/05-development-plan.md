# 05. 开发计划

## 总体节奏

建议先做 5 个阶段：

1. 文档与原型。
2. 顶部小岛壳。
3. 本地核心模块。
4. 设置与权限。
5. 外部服务与打包。

## Milestone 0：文档和项目骨架

目标：

- 完成需求、PRD、设计、架构、计划文档。
- 确定项目命名和技术栈。
- 明确不复制竞品、不破解内购的边界。

产出：

- `docs` 目录。
- 初始 README。

## Milestone 1：macOS App 壳

目标：

- 创建 Xcode macOS SwiftUI 项目。
- 实现菜单栏 App。
- 实现顶部小岛 `NSPanel`。
- 实现紧凑态/展开态状态切换。
- 实现基础定位算法。

验收：

- App 启动后顶部出现小岛。
- 鼠标悬停或快捷键可展开/收起。
- 切换 Spaces 后仍能显示。
- 重启 App 后记住基础设置。

## Milestone 2：P0 本地模块

目标：

- 时间日期模块。
- Todo 模块。
- 番茄钟模块。
- 快捷启动模块。
- 文件暂存模块。

验收：

- 无网络情况下所有 P0 模块可用。
- Todo 和配置重启不丢。
- 文件暂存支持拖入拖出。
- 番茄钟完成后发通知。

## Milestone 3：设置、快捷键和数据

目标：

- 设置窗口。
- 模块开关和排序。
- 外观配置。
- 全局快捷键。
- 导入/导出 JSON。

验收：

- 用户可关闭任意模块。
- 用户可调整小岛尺寸。
- 用户可配置展开方式。
- 数据可导出备份。

## Milestone 4：P1 外部能力

目标：

- 天气模块。
- AI 对话模块。
- 速记模块。

验收：

- API Key 存入 Keychain。
- 网络失败有错误提示。
- 未配置 API Key 时不影响其他模块。
- AI 不默认读取本地文件、笔记、Todo。

## Milestone 5：打包与优化

目标：

- 代码签名。
- Notarization。
- DMG 打包。
- 性能优化。
- 崩溃日志与反馈入口。

验收：

- Release 包可在本机安装运行。
- 空闲 CPU 接近 0。
- 常驻内存满足目标。
- 基础手工测试通过。

## 推荐任务拆分

### 第一周

- Xcode 项目初始化。
- AppDelegate 和菜单栏入口。
- IslandWindowController。
- IslandPositioner。

### 第二周

- SwiftUI 小岛 UI。
- 状态机和动画。
- 设置存储。
- 多显示器初版。

### 第三周

- Todo。
- 番茄钟。
- 通知。
- 数据持久化。

### 第四周

- 快捷启动。
- 文件暂存。
- 速记。
- 模块管理。

### 第五周

- 设置页。
- 快捷键。
- 天气。
- AI 对话。

### 第六周

- 测试。
- 性能优化。
- 打包。
- 使用文档。
