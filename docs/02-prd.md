# 02. PRD

## 产品名称

暂定：Notch Pilot

## 一句话介绍

把 Mac 顶部刘海区变成一个本地优先、无强制内购的效率驾驶舱。

## 版本范围

本文档定义 v0.1 MVP。

## 用户价值

- 少切窗口：把轻量信息放到顶部。
- 少装插件：Todo、番茄钟、快捷启动、文件架统一入口。
- 少付费焦虑：核心能力不拆成内购。
- 少数据担忧：个人数据优先本地保存。

## 顶部小岛

### 紧凑态

默认显示一个小型胶囊区域：

- 中间显示当前选中的状态模块。
- 左右两侧可显示最多两个轻量状态，例如番茄钟和天气。
- 鼠标悬停后延迟展开，延迟可配置。
- 支持快捷键展开。

### 展开态

展开后显示模块面板：

- 第一行：当前主状态、搜索/快速命令入口。
- 第二行：模块 Tab 或图标栏。
- 主区域：当前模块内容。
- 底部：设置、收起、清空类动作。

### 交互规则

- 鼠标离开后自动收起，延迟可配置。
- 正在拖拽文件时保持展开。
- 输入文字或 AI 对话时不自动收起。
- 点击小岛外部可收起。
- 全屏应用中默认隐藏，可在设置中允许显示。

## 模块需求

### 时间日期模块

- 显示时间、日期、星期。
- 可选 24 小时制。
- 可选日/周/月/年进度。

### Todo 模块

- 新增 Todo。
- 标记完成。
- 删除 Todo。
- 今日列表和全部列表。
- 本地持久化。

### 番茄钟模块

- 默认 25 分钟专注、5 分钟休息。
- 支持自定义时长。
- 开始、暂停、重置。
- 完成后通知。
- 紧凑态显示倒计时。

### 快捷启动模块

- 添加 App。
- 从小岛一键打开。
- 支持拖拽 App 到配置区。
- 支持排序、删除。

### 文件暂存模块

- 拖入文件到小岛。
- 在展开态展示文件列表。
- 支持拖出文件。
- 支持在 Finder 中显示。
- 支持清空列表。
- 默认只保存安全书签和文件引用，不复制文件本体。

### 速记模块

- 快速写入纯文本笔记。
- 最近笔记列表。
- 一键转 Todo。
- 本地搜索。

### 天气模块

- 配置城市。
- 配置 Provider 和 API Key。
- 展示当前温度、天气状态、今日最高/最低。
- 网络失败时保留最近一次缓存。

### AI 对话模块

- 支持 OpenAI-compatible endpoint。
- 用户自行配置 Base URL、API Key、模型名。
- 支持快速提问、复制回答。
- 不保存 API Key 明文，使用 Keychain。
- 默认不上传本地 Todo、笔记、文件内容。

## 设置需求

### 通用

- 开机启动。
- 主屏/所有屏幕显示。
- 全屏应用时行为。
- 展开方式：悬停、点击、快捷键。

### 外观

- 小岛宽度、高度。
- 圆角、透明度。
- 亮色/暗色/跟随系统。
- 模块图标是否显示。

### 模块管理

- 开启/关闭模块。
- 设置紧凑态主模块。
- 模块排序。

### 快捷键

- 展开/收起。
- 快速新建 Todo。
- 快速速记。
- 开始/暂停番茄钟。

### 数据

- 导出 JSON。
- 导入 JSON。
- 清空本地数据。

## v0.1 不做

- 不做在线账号系统。
- 不做内购。
- 不做破解或迁移竞品付费能力。
- 不做插件市场。
- 不做复杂多用户同步。
- 不做 App Store 上架承诺，先做本地可用和可签名分发。
