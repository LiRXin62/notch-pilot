# 07. MVP 验收标准

## 功能验收

### 顶部小岛

- App 启动后小岛显示在顶部中央。
- 有刘海屏幕能贴合刘海区域。
- 无刘海屏幕能以模拟小岛显示。
- 支持紧凑态和展开态。
- 支持鼠标悬停展开。
- 支持快捷键展开。
- 支持手动收起。

### Todo

- 可以新增 Todo。
- 可以完成 Todo。
- 可以删除 Todo。
- 重启 App 后 Todo 不丢失。

### 番茄钟

- 可以开始、暂停、重置。
- 紧凑态显示剩余时间。
- 完成后通知用户。
- 支持自定义专注和休息时长。

### 快捷启动

- 可以添加常用 App。
- 可以点击启动 App。
- 可以调整排序。
- 可以删除条目。

### 文件暂存

- 可以拖入文件。
- 可以查看暂存文件列表。
- 可以拖出文件。
- 可以清空暂存区。
- 重启后能恢复文件引用或提示文件已失效。

### 设置

- 可以开关模块。
- 可以调整小岛尺寸。
- 可以切换展开方式。
- 可以设置开机启动。
- 可以导出/导入本地数据。

## 性能验收

- 空闲 CPU 接近 0。
- 常驻内存目标低于 120MB。
- 展开动画流畅，无明显卡顿。
- 拖拽文件时没有明显延迟。
- 关闭外部模块后不会发起网络请求。

## 兼容验收

至少测试：

- MacBook 刘海屏。
- 无刘海外接屏。
- 主屏 + 外接屏。
- 亮色模式。
- 暗色模式。
- Stage Manager。
- 全屏 App。

## 隐私验收

- 首次启动不申请无关权限。
- 未启用天气/AI 时不联网。
- API Key 不明文落盘。
- 文件暂存不扫描用户目录。
- AI 模块不默认读取本地 Todo、笔记、文件。

## 失败状态验收

- 天气 API 失败时显示明确错误。
- AI API 失败时显示明确错误。
- 文件引用失效时允许移除。
- 快捷启动 App 不存在时提示重新选择。
- 权限未授予时说明如何开启。

## v0.1 完成定义

满足以下条件即可认为 v0.1 可用：

- P0 模块全部可用。
- 设置可保存。
- 重启不丢数据。
- 常用交互无崩溃。
- 在至少一台真实 Mac 上连续使用 2 小时无异常。
